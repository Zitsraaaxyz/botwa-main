## Install

From Termux

```bash
> termux-setup-storage
(after that tap on permission)
> apt update -y
> apt upgrade -y
> apt install git -y
> apt install nodejs
> git clone https://github.com/Zitsraaxyz/bot-whatsapp
> cd bot-whatsapp
> npm install
> npm start / node index.js
```

## Install
